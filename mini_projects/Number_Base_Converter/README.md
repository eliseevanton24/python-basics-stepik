# Конвертер систем счисления (Number Base Converter)

Консольная утилита на Python для перевода чисел между популярными системами счисления.

---

## Возможности

- Перевод из десятичной в двоичную, восьмеричную и шестнадцатеричную
- Перевод из двоичной, восьмеричной и шестнадцатеричной в десятичную

---

## Как работает программа

1. Пользователь вводит десятичное число — программа выводит его в трёх системах
2. Затем вводит числа в других системах — программа переводит их обратно в десятичную

## Запуск

```
python converter.py
```
