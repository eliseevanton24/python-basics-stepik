# Магический шар (Magic 8 Ball)

Консольная игра на Python, в которой магический шар отвечает на любые твои вопросы.

---

## Возможности

- 20 вариантов ответов (положительные, нейтральные, отрицательные)
- Персонализированное приветствие по имени
- Возможность задать несколько вопросов подряд

---

## Как работает программа

1. Пользователь вводит своё имя
2. Задаёт вопрос магическому шару
3. Шар даёт случайный ответ из 20 вариантов
4. Можно задать следующий вопрос или выйти

---

## Запуск
