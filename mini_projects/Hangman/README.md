# Угадайка слов! (Hangman)

Консольная игра на Python — классическая виселица с ASCII-графикой.

---

## Возможности

- Случайное слово из встроенного списка
- Угадывание по одной букве или слова целиком
- ASCII-графика виселицы (7 стадий)
- Защита от повторного ввода буквы или слова
- 6 попыток до проигрыша
- Возможность повторной игры

---

## Как работает программа

1. Программа загадывает случайное слово
2. Игрок вводит букву или слово целиком
3. При правильной букве она открывается в слове
4. При неверном вводе добавляется часть виселицы
5. Победа — угадать слово до того как закончатся 6 попыток
6. После окончания игры можно сыграть ещё раз

## Запуск

```
python hangman.py
```
