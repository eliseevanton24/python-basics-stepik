# Генератор безопасных паролей (Secure Password Generator)

Консольная утилита на Python для генерации случайных паролей с гибкими настройками.

---

## Возможности

- Генерация нескольких паролей за один запуск
- Настройка длины пароля
- Выбор набора символов:
  - цифры (`0-9`)
  - строчные буквы (`a-z`)
  - прописные буквы (`A-Z`)
  - специальные символы (`!#$%&*+-=?@^_`)
  - Исключение неоднозначных символов (`il1Lo0O`)

---

## Как работает программа

1. Пользователь выбирает количество паролей
2. Задаёт длину пароля
3. Настраивает набор символов (цифры, буквы, спецсимволы)
4. При желании исключает неоднозначные символы
5. Программа генерирует и выводит готовые пароли

## Запуск

```
python generate.py
```
