# Числовая угадайка (Number Guessing Game)

Консольная игра на Python, в которой пользователь пытается угадать случайное число.

---

## Возможности

- Генерация случайного числа в заданном диапазоне
- Проверка корректности ввода
- Подсчёт количества попыток
- Подсказки (больше / меньше)
- Возможность повторной игры

---

## Как работает программа

1. Пользователь задаёт максимальное число `n`
2. Программа загадывает число от `1` до `n`
3. Пользователь пытается угадать число
4. Программа даёт подсказки:
   - больше
   - меньше
5. После угадывания выводится количество попыток

## Запуск

```
python game.py
```
