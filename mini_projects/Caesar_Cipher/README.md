# Шифр Цезаря (Caesar Cipher)

Консольная утилита на Python для шифрования и дешифрования текста методом Цезаря.

---

## Возможности

- Шифрование и дешифрование текста
- Поддержка русского и английского языков одновременно
- Любой шаг сдвига
- Пробелы и знаки препинания сохраняются без изменений

---

## Как работает программа

1. Пользователь выбирает режим: шифрование (`e`) или дешифрование (`d`)
2. Вводит шаг сдвига `k`
3. Вводит текст
4. Программа сдвигает каждую букву на `k` позиций в алфавите
5. Выводит результат

## Запуск
```
python caesar_cipher.py
```
